## 🎨 Как использовать

### Переключение темы
1. Открыть боковое меню (кнопка с аватаром)
2. Перейти в "Настройки"
3. Секция "Кастомизация"
4. Выбрать "Тёмная" или "Светлая"

### Выбор цветовой схемы
1. В той же секции "Кастомизация"
2. Выбрать одну из 6 схем (Default, Blue, Purple, Green, Red, Orange)

### Настройка размера шрифта
1. В секции "Кастомизация"
2. Выбрать Small (14px), Medium (16px) или Large (18px)

### Настройка плотности
1. В секции "Кастомизация"
2. Выбрать Compact (0.8), Comfortable (1.0) или Spacious (1.2)

### Установка фона для чата
1. Открыть чат
2. Открыть боковое меню → Настройки → Кастомизация
3. Нажать "Загрузить фон для этого чата"
4. Выбрать изображение
5. Фон применится только к текущему чату

### Удаление фона
1. В той же секции нажать "Удалить фон"

### Сброс всех настроек
1. Нажать "Сбросить всё" внизу секции

## 📦 Технические детали

### Сохранение настроек
Все настройки сохраняются в `localStorage` под ключом `nexo-theme`:
```json
{
  "state": {
    "mode": "dark",
    "colorScheme": "default",
    "fontSize": "medium",
    "density": "comfortable",
    "chatBackgrounds": [
      {
        "chatId": "chat-123",
        "type": "image",
        "value": "https://example.com/bg.jpg"
      }
    ]
  },
  "version": 0
}
```

### Применение темы к DOM
При изменении настроек автоматически обновляются атрибуты `<html>`:
- `data-theme="dark"` или `data-theme="light"`
- `data-color-scheme="default"` (или другая схема)
- `style="font-size: 16px; --density: 1"`

### CSS переменные
Все цвета и стили используют CSS переменные, которые переопределяются в зависимости от темы:
```css
[data-theme="light"] {
  --color-surface: #ffffff;
  --color-text: #000000;
  /* ... */
}

[data-color-scheme="blue"] {
  --color-accent: #3b82f6;
  /* ... */
}
```

## 🐛 Исправленные баги

1. **Ошибка TypeScript**: `Property 'chatTheme' does not exist`
   - **Причина**: Старый API themeStore использовал `chatTheme`, новый использует `mode` и `colorScheme`
   - **Решение**: Удалены все ссылки на `chatTheme` из ChatView.tsx

2. **Неиспользуемые переменные в SideMenu**
   - **Причина**: Старые переменные `themeIndex` и `themeCards` больше не нужны
   - **Решение**: Удалены из state и кода

3. **Неправильный тип возвращаемого значения getChatBackground**
   - **Причина**: Метод возвращает объект `ChatBackground`, а не строку
   - **Решение**: Добавлена проверка `chatBackgroundObj?.type === 'image'` и извлечение `value`
