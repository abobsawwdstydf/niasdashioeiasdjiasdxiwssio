## 🔧 ТЕХНИЧЕСКИЕ ДЕТАЛИ

### Backend:
- **Framework:** Express.js + TypeScript
- **Database:** PostgreSQL (Neon)
- **ORM:** Prisma
- **AI Providers:** Cerebras, Groq, SambaNova, OpenRouter
- **Балансировка:** Round-robin с fallback
- **Очередь:** До 50 запросов при перегрузке
- **Таймаут:** 45 секунд на запрос

### Frontend:
- **Framework:** React 18 + TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **State:** Zustand
- **Real-time:** Socket.io
- **Build:** Vite
- **Bundle size:** 1.44 MB (362 KB gzipped)

### API Endpoints:

#### AI:
```typescript
POST /api/ai/context
Body: { messageId, chatId, question? }
Response: { text: string, context: string }

POST /api/ai/suggestions
Body: { chatId, lastMessage }
Response: { suggestions: string[] }

POST /api/ai/autocomplete
Body: { text }
Response: { completion: string }

POST /api/ai/grammar
Body: { text }
Response: { corrected: string, hasChanges: boolean, original: string }
```

#### Folders:
```typescript
GET /api/folders
Response: Folder[]

POST /api/folders
Body: { name, icon, color }
Response: Folder

PUT /api/folders/:id
Body: { name?, icon?, color?, order? }
Response: Folder

DELETE /api/folders/:id
Response: { success: boolean }

POST /api/folders/:id/chats
Body: { chatId }
Response: Folder

DELETE /api/folders/:id/chats/:chatId
Response: Folder
```
