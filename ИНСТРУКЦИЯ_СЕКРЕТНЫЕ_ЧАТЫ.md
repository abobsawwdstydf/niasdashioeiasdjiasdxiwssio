# 🔒 Секретные чаты - Инструкция пользователя

## Что такое секретный чат?

Секретный чат - это защищённый чат с дополнительными функциями безопасности:
- 🛡️ **End-to-End шифрование** - только вы и собеседник можете читать сообщения
- ⏰ **Самоуничтожающиеся сообщения** - автоматическое удаление через заданное время
- ⚠️ **Защита от скриншотов** - уведомление при попытке скриншота
- 🔐 **Пароль** - дополнительная защита (опционально)

---

## Как создать секретный чат?

### Шаг 1: Откройте профиль пользователя
- Нажмите на аватар пользователя в списке чатов
- Или нажмите на имя пользователя в заголовке чата

### Шаг 2: Нажмите кнопку "Секретный чат"
- Кнопка с градиентом (фиолетовый → розовый)
- Расположена под кнопкой "Написать сообщение"

### Шаг 3: Настройте параметры

#### Пароль (опционально)
- Введите пароль для защиты чата
- Оставьте пустым, если пароль не нужен
- Пароль потребуется для открытия чата

#### Таймер самоуничтожения
Выберите время, через которое сообщения будут автоматически удаляться:
- **Без таймера** - сообщения не удаляются
- **5 секунд** - для очень конфиденциальной информации
- **10 секунд** - быстрое удаление
- **30 секунд** - стандартное время
- **1 минута** - для коротких разговоров
- **5 минут** - для обычных разговоров
- **1 час** - для длительных разговоров
- **1 день** - для важных сообщений

### Шаг 4: Нажмите "Создать"
- Секретный чат будет создан
- Чат автоматически откроется
- Профиль пользователя закроется

---

## Как распознать секретный чат?

### В списке чатов
- Иконка замка 🔒 рядом с именем чата
- Цвет иконки: фиолетовый

### В заголовке чата
- Иконка замка 🔒 слева от имени чата
- Цвет иконки: фиолетовый

---

## Возможности секретного чата

### 1. End-to-End шифрование
- Включается автоматически при создании
- Только вы и собеседник можете читать сообщения
- Даже сервер не может расшифровать сообщения

### 2. Самоуничтожающиеся сообщения
- Настраивается при создании чата
- Сообщения удаляются автоматически через заданное время
- Удаление происходит на всех устройствах

### 3. Защита от скриншотов
- При попытке скриншота собеседник получит уведомление
- Работает только на мобильных устройствах
- В веб-версии недоступно (ограничение браузера)

### 4. Пароль
- Дополнительная защита чата
- Пароль потребуется для открытия чата
- Можно не устанавливать пароль

---

## Ограничения

### Технические ограничения:
- **Детекция скриншотов** - невозможна в веб-браузере
- **E2E шифрование** - требует обмена ключами (в разработке)
- **Самоуничтожение** - работает только на сервере

### Что НЕ защищено:
- Скриншоты на веб-версии
- Копирование текста
- Фотографирование экрана
- Сохранение медиафайлов

## Часто задаваемые вопросы

### Можно ли отменить создание секретного чата?
Нет, после создания секретный чат нельзя превратить в обычный. Можно только удалить чат.

### Можно ли изменить таймер самоуничтожения?
Нет, таймер устанавливается при создании чата и не может быть изменён.

### Можно ли пересылать сообщения из секретного чата?
Нет, сообщения из секретного чата нельзя пересылать.

### Можно ли добавить участников в секретный чат?
Нет, секретный чат всегда только между двумя пользователями.

### Что происходит при удалении секретного чата?
Все сообщения удаляются безвозвратно на всех устройствах.

### Можно ли восстановить удалённые сообщения?
Нет, самоуничтожившиеся сообщения восстановить невозможно.

## Советы по безопасности

### ✅ Рекомендуется:
- Использовать пароль для важных чатов
- Выбирать короткий таймер для конфиденциальной информации
- Проверять иконку замка перед отправкой сообщения
- Не делать скриншоты секретных чатов

### ❌ Не рекомендуется:
- Использовать простые пароли (123456, password)
- Оставлять устройство без блокировки
- Делиться паролем от секретного чата
- Сохранять медиафайлы из секретного чата

## Примеры использования

### Для личных разговоров
- Таймер: 5 минут
- Пароль: не требуется
- Подходит для обычных разговоров

### Для конфиденциальной информации
- Таймер: 30 секунд
- Пароль: обязательно
- Подходит для паролей, кодов, адресов

### Для важных документов
- Таймер: 1 день
- Пароль: обязательно
- Подходит для контрактов, договоров

### Для быстрых сообщений
- Таймер: 10 секунд
- Пароль: не требуется
- Подходит для одноразовых кодов
