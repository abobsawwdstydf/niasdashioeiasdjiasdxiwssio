## 4. Поток данных (Data Flow)

```
┌─────────────────┐
│  Пользователь   │
└────────┬────────┘
         │
         │ 1. Нажатие кнопки 🎥
         ▼

│  MessageInput               │
│  • mode: 'video'            │
│  • setShowRecorder(true)    │

         │
         │ 2. Открыть модалку
         ▼
┌─────────────────────────────┐
│  VideoNoteRecorder          │
│  • getUserMedia()           │

└────────┬────────────────────┘
         │
         │ 3. Запись
         ▼

│  MediaRecorder              │

│  • stop() → Blob            │

         │
         │ 4. Отправка
         ▼
┌─────────────────────────────┐
│  API: POST /api/messages/   │
│       video-note            │
│  • FormData(blob, chatId)   │
│  • → { id, videoUrl, ... }  │
└────────┬────────────────────┘
         │
         │ 5. WebSocket broadcast
         ▼
┌─────────────────────────────┐

         │
         │ 6. Рендер
         ▼
```

## 6. API Flow

```
┌─────────────────────────────────────────────────────────────┐
│  POST /api/messages/video-note                               │
└─────────────────────────────────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────┐
│  1. Валидация                                                │
│     • MIME-type (video/webm, video/mp4)                     │
│     • Размер (≤ 10 MB)                                      │
│     • Длительность (≤ 90 сек)                               │
└─────────────────────────────────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────┐
│  2. Сохранение файла                                         │
│     • Генерация уникального имени                           │
│     • Сохранение на диск/S3                                 │
│     • Путь: /uploads/video-notes/{filename}                 │
└─────────────────────────────────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────┐
│  3. Генерация thumbnail                                      │
│     • Извлечение первого кадра (ffmpeg)                     │
│     • Сохранение как JPEG                                   │
│     • Путь: /uploads/thumbnails/{filename}.jpg              │
└─────────────────────────────────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────┐
│  4. Создание записи в БД                                     │
│     • type: VIDEO_NOTE                                      │
│     • videoUrl: /api/media/video-notes/{filename}           │
│     • duration: {seconds}                                   │
│     • thumbnail: /api/media/thumbnails/{filename}.jpg       │
└─────────────────────────────────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────┐
│  5. WebSocket broadcast                                      │
│     • socket.to(chatId).emit('new_message', message)        │
└─────────────────────────────────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────┐
│  6. Response                                                 │
│     • { id, videoUrl, duration, thumbnail, ... }            │
└─────────────────────────────────────────────────────────────┘
```
