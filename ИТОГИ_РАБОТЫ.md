   - Превью папки
   - Валидация названия (макс 30 символов)

   - `createFolder()` - создать папку
   - `updateFolder()` - обновить папку

## 📝 ТЕХНИЧЕСКИЕ ДЕТАЛИ

### AI Endpoints:
- Используют существующую систему балансировки провайдеров
- Cerebras (приоритет) → Groq → SambaNova → OpenRouter
- Все запросы проходят через очередь при перегрузке
- Таймаут: 45 секунд
- Максимальный размер очереди: 50 запросов

### Folders API:
- PostgreSQL база данных (Neon)
- Prisma ORM
- Many-to-many связь между папками и чатами
- Поддержка сортировки (order field)
- Проверка прав доступа (только свои папки)

### Frontend:
- React 18 + TypeScript
- Framer Motion для анимаций
- Tailwind CSS для стилей
- Zustand для state management
- Socket.io для real-time обновлений
