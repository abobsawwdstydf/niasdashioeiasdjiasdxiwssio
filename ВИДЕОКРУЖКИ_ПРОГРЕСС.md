### MessageInput интеграция
**Что нужно сделать**:
1. Добавить состояние `mode: 'voice' | 'video'`
2. Добавить состояние `showVideoRecorder: boolean`
3. Добавить обработчик зажатия кнопки (1.3 сек)
4. Изменить иконку кнопки (🎤 ↔ 🎥)
5. Открывать VideoNoteRecorder при нажатии в режиме video

**Код для добавления**:
```typescript
const [recordMode, setRecordMode] = useState<'voice' | 'video'>('voice');
const [showVideoRecorder, setShowVideoRecorder] = useState(false);
const [pressTimer, setPressTimer] = useState<NodeJS.Timeout | null>(null);

const handleRecordButtonPress = () => {
  const timer = setTimeout(() => {
    // Toggle mode after 1.3 seconds
    setRecordMode(prev => prev === 'voice' ? 'video' : 'voice');
  }, 1300);
  setPressTimer(timer);
};

const handleRecordButtonRelease = () => {
  if (pressTimer) {
    clearTimeout(pressTimer);
    setPressTimer(null);
  }
  
  // If in video mode, open recorder
  if (recordMode === 'video') {
    setShowVideoRecorder(true);
  } else {
    // Start voice recording
    startRecording();
  }
};
```

## 🐛 Известные проблемы

1. **Thumbnail не генерируется** - Требует установки ffmpeg на сервере
2. **Нет валидации разрешения** - Сервер не проверяет разрешение видео
3. **Нет rate limiting** - Можно отправлять неограниченное количество видеокружков
4. **Нет индикации прогресса загрузки** - Пользователь не видит прогресс отправки
