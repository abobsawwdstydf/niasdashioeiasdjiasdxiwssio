# Nexo Messenger для Windows

## Описание
Настольное приложение мессенджера Nexo для Windows.

## Функции
- Авторизация через QR-код или ключ (37 символов)
- Мгновенные сообщения через WebSocket
- Голосовые и видеозвонки (WebRTC)
- Отправка файлов (всё хранится в Telegram)
- Каналы и группы
- Истории
- Настройки сервера (URL)
- Вкладка "Устройства" с информацией о сессиях

## Настройки сервера
В приложении можно указать URL сервера:
- По умолчанию: `http://localhost:3001`
- Можно изменить в настройках

## Сборка
```bash
# Установка зависимостей
npm install

# Сборка EXE
npm run build:windows

# Запуск
npm run start:windows
```

## Технологии
- Electron / Tauri
- React для UI
- Socket.IO клиент
- WebRTC для звонков
