# Nexo Messenger для Android

## Описание
Реальное Android-приложение мессенджера Nexo с поддержкой всех функций.

## Функции
- Авторизация через QR-код или ключ (37 символов)
- Мгновенные сообщения через WebSocket
- Голосовые и видеозвонки (WebRTC)
- Отправка файлов (всё хранится в Telegram)
- Каналы и группы
- Истории
- Настройки сервера (URL)

## Настройки сервера
В приложении можно указать URL сервера:
- По умолчанию: `http://localhost:3001`
- Можно изменить в настройках

## Сборка
```bash
# Установка зависимостей
npm install

# Сборка APK
npm run build:android

# Запуск в эмуляторе
npm run start:android
```

## Технологии
- React Native / Kotlin (в зависимости от версии)
- Socket.IO клиент
- WebRTC для звонков
