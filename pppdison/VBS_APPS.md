# Nexo Messenger - VBS Приложения

## 📱 Что такое VBS приложение?

VBS (VBScript) - это скриптовый язык Windows который может создавать **GUI приложения** с окнами, кнопками, формами и полноценным интерфейсом.

## 🚀 Запуск

### Главное приложение (Windows)
```
Двойной клик по: pppdison/windows/NexoMessenger.vbs
```

Откроется **полноценное GUI окно** с:
- ✅ Экраном входа и регистрации
- ✅ Чатами и сообщениями
- ✅ Настройками сервера
- ✅ Поиском по чатам
- ✅ Отправкой сообщений

### Universal Launcher
```
Двойной клик по: pppdison/launcher.vbs
```

Показывает меню выбора платформы.

## 📊 Функции VBS приложения

### Окно входа:
- Красивый градиентный дизайн
- Вкладки "Вход" и "Регистрация"
- Поле для URL сервера
- Сохранение настроек в реестр

### Окно чата:
- Список чатов слева
- Область сообщений справа
- Поле ввода сообщений
- Кнопка отправки
- Поиск по чатам
- Кнопка выхода

## 💾 Хранение настроек

Все настройки сохраняются в реестре Windows:
```
HKEY_CURRENT_USER\Software\NexoMessenger\
├── ServerURL\   - URL сервера
├── Token\       - Токен авторизации
└── User\        - Имя пользователя
```

## ⚙️ Как работает

1. **Запуск** → Открывается Internet Explorer с HTML/CSS интерфейсом
2. **Авторизация** → HTTP запросы к серверу через WinHttp
3. **Чаты** → Загрузка списка чатов и сообщений
4. **Отправка** → Отправка сообщений через POST запросы
5. **Сохранение** → Токен и настройки в реестре

## 🔧 Требования

- Windows 7/8/10/11
- Internet Explorer 11 (встроен в Windows)
- Доступ к серверу Nexo

## 🎯 Отличия от Electron

| Функция | VBS | Electron |
|---------|-----|----------|
| Размер | ~20 KB | ~150 MB |
| Установка | Нет | Требуется |
| Зависимости | Нет | Node.js |
| Скорость запуска | Мгновенно | 2-5 секунд |
| Функционал | Базовый | Полный |

## 📝 Примечание

VBS приложение использует Internet Explorer для отображения интерфейса, но это **не веб-страница** - это полноценное приложение с:
- Доступом к файловой системе
- Сохранением в реестр
- HTTP запросами
- Взаимодействием с ОС
