# Nexo Messenger для iOS

## Описание
Мобильное приложение мессенджера Nexo для iPhone.

## Функции
- Авторизация через QR-код или ключ (37 символов)
- Мгновенные сообщения через WebSocket
- Голосовые и видеозвонки (WebRTC)
- Отправка файлов (всё хранится в Telegram)
- Каналы и группы
- Истории
- Настройки сервера (URL)

## Настройки сервера
В приложении можно указать URL сервера:
- По умолчанию: `http://localhost:3001`
- Можно изменить в настройках

## Сборка
```bash
# Установка зависимостей
npm install

# Сборка IPA
npm run build:ios

# Запуск в симуляторе
npm run start:ios
```

## Технологии
- React Native
- Socket.IO клиент
- WebRTC для звонков
