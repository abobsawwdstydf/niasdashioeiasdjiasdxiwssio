# Nexo Messenger - Windows Desktop Installer

## 📦 Что включено:

### Установщик:
- ✅ **Тёмная тема** в стиле мессенджера
- ✅ **Выбор директории** установки
- ✅ **Ярлык на рабочем столе**
- ✅ **Ярлык в меню Пуск**
- ✅ **Деинсталлятор**
- ✅ **Поддержка русского языка**

## 🚀 Как собрать:

### Быстрая сборка:
```bash
build-desktop.bat
```

### Ручная сборка:
```bash
cd desktop
npm install
npm run build
```

## 📍 Результат:

**Файл установщика:**
```
desktop\dist\Nexo-Messenger-Setup-1.0.0.exe
```

**Размер:** ~80-100 MB

## 💻 Установка:

1. **Запусти** `Nexo-Messenger-Setup-1.0.0.exe`
2. **Выбери папку** для установки (по умолчанию: `C:\Program Files\Nexo Messenger`)
3. **Нажми Install**
4. **Готово!** Ярлык появится на рабочем столе

## 🎨 Особенности установщика:

- **Тёмный дизайн** в стиле Nexo
- **Выбор языка** (English/Русский)
- **Можно изменить** папку установки
- **Создаёт** ярлыки автоматически
- **Чистое удаление** через Панель управления

## 🔧 Требования:

- **ОС:** Windows 7/8/10/11 (64-bit)
- **RAM:** 2 GB
- **Место:** 200 MB
- **.NET:** 4.5+
