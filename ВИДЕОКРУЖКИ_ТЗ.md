# 🎥 Техническое задание: Видеокружки (Video Notes)

## 1. Общее описание

Пользователь записывает **видеокружок** (до 90 сек) с передней или тыловой камеры, **без звука**. Видео автоматически воспроизводится в чате по кругу, зациклено, звук включается нажатием.

---

## 2. Интерфейс

### 2.1 Запись

#### Кнопка переключения режима
- **Расположение**: рядом с полем ввода (иконка микрофона/видео)
- **Переключение режима**: зажатие иконки **1.3 секунды**
- **Режимы**:
  - 🎤 Голосовое сообщение (по умолчанию)
  - 🎥 Видеокружок

#### Запуск записи
- **Действие**: простое нажатие на иконку (в режиме видеокружка)
- **Результат**: открывается модальное окно с видоискателем

#### Во время записи

**Видоискатель**:
- Круглый видоискатель (поток с камеры)
- Размер: 480×480px
- Форма: круг (border-radius: 50%)
- Поток в реальном времени

**Таймер**:
- Формат: `mm:ss`
- Максимум: `01:30` (90 секунд)
- Цвет: белый с тенью
- Позиция: над видоискателем

**Кнопка переключения камеры**:
- Иконка: 🔄 (циклическая стрелка)
- Позиция: правый верхний угол видоискателя
- Действие: циклическое переключение камер
  - Фронтальная → Тыловая → Фронтальная → ...
- Альтернатива: выпадающий список с выбором конкретной камеры

**Кнопка записи**:
- Большая красная кнопка в центре снизу
- Анимация пульсации во время записи
- Повторное нажатие → остановка + отправка

**Жесты**:
- ⬅️ **Смахивание влево/вправо** → отмена записи
- ⬆️ **Смахивание вверх** → закрепить запись (продолжать без повторного нажатия)

**Индикаторы**:
- Красная точка "REC" в углу
- Прогресс-бар (круговой вокруг видоискателя)

#### Остановка записи
- **Действие**: повторное нажатие на кнопку записи
- **Результат**: остановка + автоматическая отправка

#### Отмена записи
- **Действие**: смахивание влево или вправо по видоискателю
- **Результат**: закрытие модального окна, удаление записи

#### Закрепление записи
- **Действие**: смахивание вверх по видоискателю
- **Результат**: запись продолжается без необходимости держать кнопку

---

### 2.2 Воспроизведение в чате

#### Внешний вид
- **Форма**: круглое превью (border-radius: 50%)
- **Размер**: 200×200px в чате
- **Рамка**: тонкая градиентная рамка (синий → фиолетовый)
- **Иконка**: 🎥 в центре (до воспроизведения)

#### Поведение
- **Автовоспроизведение**: при скролле в зону видимости (IntersectionObserver)
- **Зацикливание**: `loop` атрибут
- **Звук**: выключен по умолчанию (`muted`)
- **Нажатие на кружок**:
  - 1-й клик → включить звук
  - 2-й клик → пауза
  - 3-й клик → продолжить
  - 4-й клик → выключить звук

#### Индикаторы
- **Иконка звука**: 🔇 (выкл) / 🔊 (вкл) в правом верхнем углу
- **Прогресс**: тонкая круговая полоса вокруг видео
- **Пауза**: иконка ⏸ в центре (при паузе)

---

### 2.3 Управление сообщением

#### Контекстное меню (правый клик / долгое нажатие)
- Ответить
- Переслать
- Удалить
- Сохранить видео
- Копировать ссылку

#### Реакции
- Стандартные эмодзи-реакции под сообщением

## 3. Технические требования (веб)

### 3.1 Запись

#### API
```javascript
navigator.mediaDevices.getUserMedia({
  video: {
    facingMode: 'user', // или 'environment'
    width: { ideal: 480 },
    height: { ideal: 480 },
    frameRate: { ideal: 30, max: 30 }
  },
  audio: false // ВСЕГДА false
})
```

#### MediaRecorder
```javascript
const mediaRecorder = new MediaRecorder(stream, {
  mimeType: 'video/webm;codecs=vp9', // или 'video/mp4' если поддерживается
  videoBitsPerSecond: 750000 // 0.75 Mbps
});
```

#### Параметры записи
- **Камера**: передняя или тыловая (выбор пользователя)
- **Переключение камер**: остановка старого трека, запуск нового
- **Макс. длительность**: 90 секунд (автостоп)
- **Формат**: WebM (VP9) или MP4 (H.264)
- **Разрешение**: 480×480px
- **FPS**: 24-30
- **Битрейт**: 0.5-1 Mbps
- **Звук**: ВСЕГДА отключён

#### Обработка после записи
1. **Обрезка до круга**: CSS `clip-path: circle(50%)`
2. **Сжатие**: через MediaRecorder с низким битрейтом
3. **Создание blob**: `new Blob(chunks, { type: 'video/webm' })`
4. **Отправка**: `multipart/form-data` на сервер

```html
  </div>

```

#### CSS
```css
.video-note {
  width: 200px;
  height: 200px;
  border-radius: 50%;
  object-fit: cover;
  clip-path: circle(50%);
}
```

#### JavaScript
```javascript
// Автовоспроизведение при скролле
const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      entry.target.play();
    } else {
      entry.target.pause();
    }
  });
}, { threshold: 0.5 });

observer.observe(videoElement);
```

#### Управление
- **play()** / **pause()** по нажатию
- **muted = false** / **muted = true** по нажатию
- **loop = true** всегда
- **Прогресс**: обновление через `timeupdate` event

### 3.3 Offline поддержка

#### Service Worker
```javascript
// Кэширование видео
self.addEventListener('fetch', (event) => {
  if (event.request.url.includes('/api/media/video-notes/')) {
    event.respondWith(
      caches.match(event.request).then(response => {
        return response || fetch(event.request).then(fetchResponse => {
          return caches.open('video-notes-v1').then(cache => {
            cache.put(event.request, fetchResponse.clone());
            return fetchResponse;
          });
        });
      })
    );
  }
});
```

#### Очередь отправки (IndexedDB)
```javascript
// Сохранение в очередь при отсутствии сети
if (!navigator.onLine) {
  await db.videoNotesQueue.add({
    blob: videoBlob,
    chatId: chatId,
    timestamp: Date.now()
  });
}

// Отправка при восстановлении сети
window.addEventListener('online', async () => {
  const queue = await db.videoNotesQueue.toArray();
  for (const item of queue) {
    await sendVideoNote(item.blob, item.chatId);
    await db.videoNotesQueue.delete(item.id);
  }
});
```

### 3.4 Обработка ошибок

#### Нет камеры
```javascript
try {
  const stream = await navigator.mediaDevices.getUserMedia({ video: true });
} catch (error) {
  if (error.name === 'NotFoundError') {
    alert('Камера не найдена. Подключите камеру и попробуйте снова.');
  }
}
```

#### Нет разрешения
```javascript
if (error.name === 'NotAllowedError') {
  alert('Разрешите доступ к камере в настройках браузера.');
}
```

#### Камера занята
```javascript
if (error.name === 'NotReadableError') {
  alert('Камера используется другим приложением. Закройте его и попробуйте снова.');
}
```

```
┌─────────────────────────┐
│  Режим: Голосовое       │
│  Иконка: 🎤             │
└───────────┬─────────────┘
            │
            │ Зажатие 1.3 сек
            ▼
┌─────────────────────────┐
│  Режим: Видеокружок     │
│  Иконка: 🎥             │
└───────────┬─────────────┘
            │
            │ Простое нажатие
            ▼
┌─────────────────────────┐
│  Запись видеокружка     │
│  • Видоискатель         │
│  • Таймер (00:00)       │
│  • Кнопка камеры        │
│  • Кнопка записи        │
└───────────┬─────────────┘
            │
            ├─ Повторное нажатие → Остановка + Отправка
            ├─ Смахивание вверх → Закрепить (продолжать)
            ├─ Смахивание влево/вправо → Отмена
            ├─ Таймер > 90 сек → Автостоп + Отправка
            └─ Кнопка камеры → Переключение источника
```

## 6. API Endpoints

### 6.1 Отправка видеокружка

**POST** `/api/messages/video-note`

**Request**:
```
Content-Type: multipart/form-data

chatId: string
video: File (blob)
duration: number (секунды)
```

**Response**:
```json
{
  "id": "msg-123",
  "type": "video_note",
  "chatId": "chat-456",
  "senderId": "user-789",
  "videoUrl": "https://cdn.nexo.com/video-notes/abc123.webm",
  "duration": 45,
  "thumbnail": "https://cdn.nexo.com/thumbnails/abc123.jpg",
  "createdAt": "2026-04-19T12:00:00Z"
}
```

### 6.2 Получение видеокружка

**GET** `/api/media/video-notes/:id`

**Response**: Binary video file (WebM or MP4)

## 8. Производительность

### 8.1 Оптимизация записи
- Битрейт: 0.5-1 Mbps (баланс качество/размер)
- Разрешение: 480×480 (не больше)
- FPS: 24-30 (не 60)
- Формат: WebM (VP9) - лучшее сжатие

### 8.2 Оптимизация воспроизведения
- Ленивая загрузка (IntersectionObserver)
- Кэширование в Service Worker
- Preload: metadata (не auto)
- Thumbnail для превью

### 8.3 Размер файлов
- 10 сек: ~0.5-1 MB
- 30 сек: ~1.5-3 MB
- 60 сек: ~3-6 MB
- 90 сек: ~4.5-9 MB

## 9. Безопасность

### 9.1 Валидация на сервере
- Проверка MIME-type (video/webm, video/mp4)
- Проверка длительности (≤ 90 сек)
- Проверка размера файла (≤ 10 MB)
- Проверка разрешения (≤ 1080×1080)

### 9.2 Защита от злоупотреблений
- Лимит: 10 видеокружков в минуту
- Лимит: 100 видеокружков в день
- Антиспам фильтр
